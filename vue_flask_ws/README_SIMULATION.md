# AI 缺陷检测与对话应用

## 当前状态

由于Windows环境下Flask服务器启动存在系统兼容性问题，应用目前使用智能模拟数据来演示完整功能。

## 功能特性

### 🔍 缺陷检测模式
- **动态提示词响应**: 根据选择的提示模板生成不同的检测结果
  - `简单提示`: 基础检测结果
  - `JSON格式提示`: 详细的JSON结构化结果
  - `详细提示`: 包含分析说明的详细报告

- **智能缺陷生成**: 每个提示词产生不同的缺陷类型和位置
- **实时可视化**: Canvas绘制缺陷边界框和标签
- **响应式布局**: 适配1920x1080大屏幕

### 💬 AI对话模式
- **上下文感知**: 根据用户输入动态生成响应
- **多模态支持**: 支持纯文本和图片+文本对话
- **智能回复**: 多个预设响应随机选择，避免重复
- **消息历史**: 完整的对话记录和时间戳

## 技术实现

### 前端 (React + TypeScript + Vite)
- 模式切换界面
- 智能模拟API响应
- Canvas缺陷可视化
- 响应式设计

### 后端 (Flask + Python)
- 完整的API架构 (暂时使用模拟数据)
- CORS跨域支持
- 错误处理机制

## 使用方法

1. **启动前端**: `npm run dev` (http://localhost:5173)
2. **缺陷检测**:
   - 选择模式 → 上传图片 → 选择提示词 → 检测
   - 观察不同的检测结果和可视化效果
3. **AI对话**:
   - 切换到对话模式 → 输入消息 → 发送
   - 可选上传图片进行多模态对话

## 模拟数据特性

- **缺陷检测**: 根据提示词生成不同的缺陷结果
- **对话响应**: 根据输入内容生成相关回复
- **网络延迟**: 模拟真实的API响应时间
- **错误处理**: 完整的错误状态管理
